# README

## О проекте

Этот проект — многостраничный сайт, созданный с использованием HTML и CSS.  
Основная цель проекта — разработать удобный и визуально аккуратный веб-сайт с простой навигацией между страницами и единым стилем оформления.

Сайт состоит из 5 страниц:

- Главная
- Участники
- О проекте
- Журнал
- Ресурсы

---

## Используемые технологии

- HTML5
- CSS3

---

## Структура сайта

### Главная страница

На главной странице размещена основная информация о сайте, приветственный блок и навигация по разделам.  
При разработке использовалась структура HTML с разделением контента на секции и блоки.

### Страница «Участники»

На странице команды представлены участники проекта.  
Для оформления карточек участников использовались CSS-стили:

- flexbox для расположения элементов;
- отступы и выравнивание;
- оформление текста.

### Страница «О проекте»

На этой странице описана цель создания сайта и его особенности.  
Для текста были настроены:

- размеры шрифтов;
- интервалы;
- цветовая палитра;
- адаптивное расположение блоков.

### Страница «Журнал»

Страница журнала содержит записи или новости проекта.  
Для оформления использовались:

- контейнеры;
- стилизация заголовков;
- разделение контента на отдельные блоки.

### Страница «Ресурсы»

На странице ресурсов размещены полезные ссылки и материалы.  
Для ссылок были добавлены CSS-эффекты:

- изменение цвета при наведении;
- стили кнопок;
- оформление списков.

---

## Как создавался проект

### HTML

Для каждой страницы был создан отдельный HTML-файл.  
В процессе работы использовались:

- заголовки;
- абзацы;
- списки;
- ссылки;
- контейнеры `div`;
- семантические теги.

Навигация между страницами реализована через меню с гиперссылками.

### CSS

Для оформления сайта был создан отдельный CSS-файл со стилями.  
В работе использовались:

- flexbox;
- цвета и фоны;
- отступы;
- стили текста;
- эффекты наведения;
- оформление блоков и кнопок;
- адаптивные элементы интерфейса.

Все страницы оформлены в едином стиле для поддержания целостного дизайна сайта.
