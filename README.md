## Описание проекта

Проект представляет собой приложение для автоматизированного переноса музыкальных данных между различными стриминговыми платформами, такими как Spotify, VK Музыка и другие сервисы.
Цель проекта — упростить процесс переноса плейлистов и музыкальных коллекций между платформами без потери структуры и метаданных.
