Tune My Music

tunemymusic.com →

Что взяли:

  • Анализ интерфейса и пользовательского пути при переносе плейлистов
  • Изучение списка поддерживаемых платформ (Spotify, Apple Music, YouTube Music, Deezer)
  • Принципы отображения прогресса переноса и обработки ошибок
  • Идеи для дизайна главной страницы и карточек сервисов

🔍 Tune My Music — один из самых популярных сервисов для переноса плейлистов. Spotify даже интегрировал его в своё приложение как встроенный инструмент импорта [citation:9][citation:10].

🎧

Soundiiz

soundiiz.com →

Что взяли:

  • Изучение работы с API различных стриминговых сервисов [citation:8]
  • Понимание ограничений: 200 треков на плейлист в бесплатной версии
  • Принципы синхронизации и обновления плейлистов между сервисами
  • Модель монетизации (freemium) и структуру тарифов

🔍 Soundiiz существует с 2013 года и за это время перенёс более 220 миллионов плейлистов [citation:8]. Это ценный источник знаний о масштабировании подобных сервисов.

📱

Free Your Music

freeyourmusic.com →

Что взяли:

  • Поддержка отечественных сервисов — Яндекс.Музыка и ВКонтакте [citation:4]
  • Функция автоматической синхронизации каждые 15 минут
  • Механизм «одного клика» для исправления несовпадающих треков
  • Кроссплатформенность (iOS, Android, Windows, macOS, Linux)

🔍 Free Your Music — один из немногих сервисов, который бесплатно работает с Яндекс.Музыкой и ВКонтакте [citation:4]. Для нас это важный референс, так как мы ориентируемся на российских пользователей.

GitHub (Open Source)

github.com →

Что взяли:

  • Изучение архитектуры open-source проектов для переноса плейлистов [citation:1][citation:3][citation:6]
  • Понимание OAuth-авторизации через Spotify Developer Dashboard [citation:1]
  • Примеры работы с YouTube Music API (через headers_auth.json) [citation:1]
  • Архитектурные решения: Node.js + Express, AWS Lambda, PostgreSQL, SQS [citation:6]
  • Использование библиотек: spotipy, ytmusicapi для Python [citation:1]

🔍 На GitHub есть готовые реализации переноса плейлистов. Изучение их кода помогло нам избежать типичных ошибок при проектировании собственной архитектуры [citation:1][citation:3][citation:6].

SongShift

songshift.com →

Что взяли:

  • Изучение автоматической синхронизации плейлистов между сервисами [citation:7]
  • Функция «умного сопоставления» треков (smart matching)
  • Принципы работы мобильного приложения (iOS-first подход)
  • Механизмы объединения нескольких плейлистов в один

🔍 SongShift интегрирован с Apple Music как официальный инструмент импорта плейлистов [citation:7]. Это показывает, что даже крупные игроки используют сторонние решения.
